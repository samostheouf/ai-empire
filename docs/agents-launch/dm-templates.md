## General DM Best Practices

### DO:
✓ Personalize every message
✓ Reference something specific about them
✓ Lead with value, not pitch
✓ Keep it short (50-100 words)
✓ Include a clear CTA
✓ Follow up once (not more)
✓ Respect "no" gracefully

### DON'T:
✗ Send generic copy-paste messages
✗ Pitch immediately
✗ Be pushy or aggressive
✗ Follow up more than once
✗ Take rejection personally
✗ Spam multiple channels
✗ Forget to provide value

### Follow-Up Template (if no response after 7 days):

Hey [Name],

Just circling back on my previous message about [topic].

Totally understand if now isn't the right time. Just wanted to make sure you saw it in case it's relevant.

Either way, keep crushing it! 🙌

[Your Name]
