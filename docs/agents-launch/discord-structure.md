## Rules

```
📋 COMMUNITY RULES

1️⃣ BE RESPECTFUL
• Treat everyone with respect
• No harassment, hate speech, or discrimination
• Disagreements are fine; personal attacks are not

2️⃣ STAY ON TOPIC
• Keep discussions relevant to AI, automation, and business
• Off-topic conversations belong in #general
• No spam, self-promotion, or unsolicited DMs

3️⃣ HELP OTHERS
• Share knowledge and experiences
• Be patient with newcomers
• Answer questions when you can

4️⃣ NO COMPETITOR PROMOTION
• Don't promote competing products
• Don't poach community members
• Focus on helping, not selling

5️⃣ PROTECT PRIVACY
• Don't share others' private information
• Don't share proprietary business data
• Respect confidentiality

6️⃣ USE CHANNELS APPROPRIATELY
• #help: Technical support and troubleshooting
• #general: Open discussion
• #wins: Success stories
• #feature-requests: Product suggestions

7️⃣ LISTEN TO MODERATORS
• Moderators have final say
• Respect their decisions
• Report issues to @moderators

⚠️ VIOLATIONS RESULT IN:
• Warning → Mute → Kick → Ban
• Severe violations = immediate ban
```

## Roles

```
👑 ROLES

@everyone — Default role (all members)

🎯 COMMUNITY ROLES:
• @member — Verified community member
• @active — Active contributor (100+ messages)
• @power-user — Advanced user (50+ agents deployed)

🛠️ STAFF ROLES:
• @moderator — Community moderator
• @support — Support team member
• @team — NeuraAPI team member
• @founder — Server founder

🌟 SPECIAL ROLES:
• @beta-tester — Early access to new features
• @ambassador — Community champion
• @expert — Domain expert (granted by team)

📌 ROLE PERMISSIONS:
• @everyone: Read channels, limited posting
• @member: Full access, can create threads
• @active: Can create events, boost server
• @moderator: Can manage messages, kick members
• @support: Can manage tickets, pin messages
• @team: Full admin access
```

## Bot Configuration

### 1. Welcome Bot (MEE6 or Carl-bot)
```
Settings:
• Auto-role: @member on join
• Welcome message: Custom message in #welcome
• DM welcome: Send DM with getting started guide
• Verification: React to verify in #rules
```

### 2. Moderation Bot (Dyno or AutoMod)
```
Settings:
• Auto-mod: Filter spam, profanity, links
• Logging: Audit logs in #mod-logs
• Commands: !kick, !ban, !mute, !warn
• Auto-response: Common questions
```

### 3. Utility Bot (Custom or Carl-bot)
```
Settings:
• Ticket system: Create support tickets
• Reaction roles: Self-assign roles
• Reminders: Event and announcement reminders
• Pinned messages: Auto-pin important info
```

### 4. Analytics Bot (Statbot or Custom)
```
Settings:
• Member tracking: Join/leave stats
• Activity tracking: Message frequency
• Channel stats: Most active channels
• Growth metrics: Weekly/monthly reports
```

## Moderation Guidelines

```
HANDLING ISSUES:

1. Minor (spam, off-topic):
   • Delete message
   • Warn user via DM
   • Log in #mod-logs

2. Moderate (harassment, self-promotion):
   • Delete message
   • Mute user 24 hours
   • DM warning
   • Log in #mod-logs

3. Severe (hate speech, threats, doxxing):
   • Delete message
   • Ban user immediately
   • Log in #mod-logs
   • Report to Discord Trust & Safety

MODERATOR RESPONSIBILITIES:
• Check #mod-logs daily
• Respond to reports within 24 hours
• Document all actions taken
• Escalate severe issues to @team
```
